## Visão Geral

Uma página de Login com opção de tema claro e escuro. Com um simples clique é possível alterar o tema.

## Funcionalidades

- Login.
- Tema Claro e Escuro.
- O botão abaixo da título "login" é capaz de alterar o tema da página.
